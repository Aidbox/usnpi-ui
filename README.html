Resource search app

This is a demonstration app showcasing indexed search capabilities of the Aidbox platform performed on the US NPI registry powered by Aidbox.

The app provides search on Practitioner and Organization FHIR resources.

Dev

  • Install dependencies bash npm i
  • Prepare environment bash cp env-tpl .env
  • Start docker-compose with Aidbox Dev bash docker-compose up -d
  • Register app bash npm run dev
  • Start ui bash npm start

Production

https://usnpi.netlify.com/

# Resource search app

This app lets you search through Practitioner and Organization resources

## Dev
- install deps `npm i`
- prepare envs `cp env-tpl .env`
- start docker-compose with aidbox dev `docker-compose up -d`
- start aidbox-app `npm run app`
- start ui `npm run ui`
